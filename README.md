# Gollum
A desktop solution to crud admin

## setup node environment

### linux 
```
sudo apt install npm nodejs-legacy
```
### windows
download and run node+npm installer from https://nodejs.org/en/

## how to run
```
git clone https://github.com/measdot/gollum.git
cd gollum
npm install
npm start 
#or electron .
```
## troubleshooting 
```
npm install npm -g
npm cache clean 
```
